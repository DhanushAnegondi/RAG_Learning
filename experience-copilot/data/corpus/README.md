# Your experience corpus

Drop your own documents here as `.md`, `.txt`, or `.pdf`:
- Project write-ups, architecture notes, achievements, runbooks
- Resume / bio paragraphs
- Company research notes (for "why this company" type questions)

Then run `python -m copilot.cli ingest` to (re)build the index.

These files are **gitignored** (only this README is tracked) — your real content stays local.
